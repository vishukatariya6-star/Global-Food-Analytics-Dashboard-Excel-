# Global Food Analytics Dashboard

## Project Overview
This project focuses on analyzing global food sales data and building an interactive Excel dashboard to visualize trends across different countries and product categories.

The dashboard helps in understanding global food demand and category performance.

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Charts and Visualizations

## Project Objectives
- Analyze global food sales trends
- Identify top performing regions
- Compare product category performance
- Visualize sales distribution across countries

## Data Preparation
The dataset was prepared for analysis through:

- Removing missing values
- Organizing columns
- Standardizing data formats

## Dashboard Components
The dashboard includes:

- Country-wise sales comparison
- Category-wise sales distribution
- Revenue trends visualization
- Key performance indicators

## Key Insights
- Identified regions with the highest food sales
- Observed category demand patterns
- Highlighted top performing food categories

## Skills Demonstrated
- Data visualization
- Dashboard design
- Data cleaning
- Business data interpretation

## Outcome
The project demonstrates how dashboards can help visualize global market trends and support strategic business decisions.
